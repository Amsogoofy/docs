Access to Leafy is managed through an application process to ensure service quality and security. While we maintain these standards for all servers, we may extend direct invitations to select communities at our discretion.

## How To Join The Waiting List

<Steps>
  <Step title="Checking Server Requirements" icon="discord" stepNumber={1}>
    - **500+ active members** (30-day activity)
    - **90+ days old** server age
    - **2FA enabled** on all admin accounts
    - **No ToS violations** or blacklist history
    - **1 application per server** only
  </Step>
  <Step title="Mandatory Steps" icon="list-ul" iconType="duotone" stepNumber={2}>
    1. Join our Discord server (required)
    2. Submit server information with activity proof
    3. Wait 3-7 business days for verification
    4. Receive DM approval in Discord
  </Step>
  <Step title="Application" icon="browser" iconType="duotone" stepNumber={3}>
    Fill [here](https://docs.google.com/forms/d/e/1FAIpQLSfT0D9wFSQgwWtCBTgXD_9PB6xfsLr7_2GD77lpAEzNIjP-AA/viewform?usp=sharing&ouid=101684507790335765584) the application in if you meet those requirements.
  </Step>
</Steps>
